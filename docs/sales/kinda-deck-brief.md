## 0. このデッキの目的・前提

- **誰に**：結婚相談所のオーナー／意思決定者。BELMA・IBJ・BIU 等に既に掲載中の所も対象。
- **役割**：コールドメール → 返信 → 「詳しく」メールに**抜粋5枚**を添付 → 面談で**フル10枚**をウォークスルー。
- **1ソース運用**：フル10枚を作り、★印の5枚を「メール添付用 抜粋」として切り出せるようにする（同一ソースから2書き出し）。
- **今フェーズの狙い**：完璧な営業ではなく「初見の相談所に見せてチグハグ（反論・誤解ポイント）を洗い出す」初版。
- **外部送信は保留中**：会社名・特商法・ドメイン・送信元は**プレースホルダ表記**（`[会社名]` `[所在地]` `[代表者氏名]` 等）で作る。実値差替は法人設立・ドメイン取得後。

## 1. トーン・ルール（厳守）

- **婚活／結婚相談所／お見合い／送客 は普通に使ってよい**（B2B 文書であり、ユーザー向けブランドコピーではないため。CLAUDE.md §2 の線引き）。
- ただし **CLAUDE.md §3 は守る**：焦らせない・比較で煽らない・空虚な共感をしない・**絵文字なし**・断定より寄り添い。
- 競合は**名指しでけなさない**。「Kinda はこう設計しています」という構文で事実ベースの差別化に留める。
- **「中立」という語は使わない**。自称は「ユーザーファーストな口コミ・レビュー型プラットフォーム」。
- LGBTQ+ 包摂：「異性」「男女」を当然にせず、必要なら「相手」「ふたり」で書く。

## 2. ビジュアル仕様

- ミニチュアクレイ風ベージュ。背景 `#F5EEE6`、アクセント `#D4A090`（ダスティローズ／テラコッタ）。
- 柔らかいマット質感・丸み・上からの柔らかい光。旧ゴールド（#C8A97A）路線は使わない。
- ロゴ・装飾は既存の clay アセット（`public/images` 配下）を流用可。
- 比率 **16:9**。**印刷／PDF 書き出しに耐える**こと（メール添付・面談画面共有の両対応）。

## 3. 技術メモ（Code 向け）

- **単一の自己完結 HTML**（外部依存最小）で作る。キーボード／タップでスライド送り。
- このアプリ本体（Next.js 16 / Tailwind v4）への組み込みは不要。`docs/sales/` 配下に独立スライドとして置く想定でよい。
- 各スライドは「見出し1行＋本文/箇条書き少なめ」。1スライド1メッセージ。文字を詰め込まない（ブランドが“静か”なので余白を活かす）。

## 4. スライド構成（フル10枚 / ★＝メール用抜粋5枚）

### 1. 表紙 ★
- **主要メッセージ**：結婚相談所を「カウンセラーの口コミ」で選ぶ、新しい掲載プラットフォーム。
- 入れる要素：サービス名 Kinda ／サブコピー「@cosme・食べログの結婚相談所版」／「結婚相談所さま向け ご提案」。
- ビジュアル：clay ベージュ全面、ロゴ中心、静かな第一印象。

### 2. 課題提起
- **主要メッセージ**：未入会者は「どの相談所か」より「信頼できる人に出会えるか」で迷っている。
- コピー方向：既存の比較サイトは“相談所単位＋広告依存”の構造で、いい担当者がいても埋もれてしまう。※競合名は出さず、構造の話として淡々と。

### 3. Kinda の答え（差別化の核）★
- **主要メッセージ**：カウンセラー個人の、面談を完了した人だけが書ける認証済み口コミで選ばれる。
- 3本柱：①カウンセラー単位で選べる ②面談完了者だけの認証口コミ（成りすまし不可） ③出会い〜予約まで一気通貫。

### 4. 届く未入会者の“質”（体験フロー）
- **主要メッセージ**：診断（Kinda type）→ 口コミで確認 → その場で予約。御社には“検討済みで温度の高い未入会者”が届く。
- 補足：数より質。あらゆる「ふたり」を前提とした設計（LGBTQ+ 包摂）を1行添える。

### 5. 料金 ★
- **主要メッセージ**：掲載費・初期費用ゼロ。送客料 **¥5,000／件の成果報酬のみ**。予約が来なければ費用ゼロ。
- 添える条件：契約縛り・違約金なし。掲載停止・再開はいつでも。
- 演出：数字を大きく、シンプルに。最大の安心材料。

### 6. 課金設計＝前払い＋情報ロック ★
- **主要メッセージ**：“¥5,000 払って空振り”が構造的に起きない仕組み。
- フロー図で：予約確定＋24hキャンセル猶予を過ぎて初めて請求 → 入金確認後に本名・連絡先を開示。
- 二層開示：入金前は「メッセージ・希望日時・ニックネーム・テーマ」（準備に足る分）／入金後に「本名・連絡先・詳細」。
- 返金条件：24h以内キャンセル・相談所都合・日程変更合意は返金。

### 7. 信頼性とコンプライアンス
- **主要メッセージ**：口コミは面談完了者のみ（専用URL＋認証コード）＝成りすまし不可。
- 代理掲載：口コミゼロでも既存の体験談を運営が代理入力。「代理掲載」バッジで透明性を担保（景品表示法・ステマ規制対応）。

### 8. 導入はかんたん＋早期参加メリット
- **主要メッセージ**：やることは「掲載するだけ」。運用は管理画面で予約・口コミ・通知を確認するだけ。
- 早期参加：先行掲載の相談所には優遇措置。将来は送客実績に応じた定額掲載プランも選べる。
- 拡張ビジョン：婚活準備 → お見合い → デート → その先まで広がるプラットフォーム（早く入るほど有利）。

### 9. よくあるご懸念（3つだけ）
- **主要メッセージ**：気になる点に先回りで答える。
  - ステマでは？ → 面談認証で構造的に不可。
  - BELMA 等と何が違う？ → 相談所単位の広告順位ではなく、カウンセラー個人の実体験＋予約導線。**今の掲載先はそのまま、併用できる**。
  - 予約が来なかったら？ → 費用ゼロ。

### 10. 次の一歩（CTA）★
- **主要メッセージ**：まずは掲載（無料）から。
- アクション：お手続きは最短◯分。`[partners URL]`（暫定 `https://my-app-rp9u.vercel.app/partners`）からお申し込み、または面談で詳細を。
- トーン：煽らない言い切り。
