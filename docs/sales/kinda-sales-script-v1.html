2〜15分

自
これまでの集客サービスと一番違うのは、「相談所単位」ではなく「カウンセラー個人単位」で選ばれるという点です。ユーザーにとっては、どの人に相談するかが一番大事なので、そこに口コミをつけました。

Hook — Kinda voices 取材という切り札

自

※ この切り札は全員に必ず出すものではない。「ぜひこの人を載せたい」と思える相手にだけ出す。安売りしない。

03
Objection Handling
反論別・切り返し集
Q
既存のサービス（ゼクシィ等）で集客できているので、今は必要ないです。
それは素晴らしいですね。Kinda は既存の集客を置き換えるものではなく、「口コミで比較したい」層への別チャネルです。初期費用ゼロなので、試してみて予約が来なければ費用はかかりません。並走させていただく形でご検討いただけませんか。
Q
他の口コミ・比較サイト（みん婚など）と何が違うんですか？
大きく2つです。1つは、「相談所」ではなく「カウンセラー個人」に口コミが付くこと。ユーザーは結局「どの人に相談するか」で選ぶので、そこに評価をつけました。もう1つは、口コミを書けるのが Kinda 経由で実際に面談した人だけという点です。やらせや業者の投稿が構造上できないので、口コミ自体の信頼度が違います。既存サイトと置き換えるのではなく、並走できるチャネルとお考えください。
Q
掲載するのに、こちらは何を準備すればいいんですか？手間がかかりませんか？
ご準備いただくのはプロフィールの要点と写真を数点お送りいただくだけです。ページへの入力やレイアウトはKinda 運営が代行します。既存の体験談があれば、それも代理掲載で載せられます。運用が始まってからも、管理画面で予約と口コミを確認いただくだけで、日々の手間はほとんどかかりません。
Q
うちは小さい相談所なので、口コミがありません。選ばれないのでは？
Kinda には代理掲載という仕組みがあります。これまでの実績（自社サイトの体験談・お客様からいただいた感想など）をテキストでお送りいただければ、Kinda 運営が代わりに入力して掲載します。口コミゼロでも、すぐに実績を見せられる状態でスタートできます。
Q
悪い口コミを書かれたら困ります。
口コミは、Kinda 経由で実際に面談を完了したユーザーにのみ発行される認証コードを使って初めて投稿できます。第三者の成りすましは構造上不可能です。また、誹謗中傷・虚偽はガイドライン違反として審査・削除します。相談所側から返信（公開コメント）を付けることもできます。
Q
¥5,000 という金額の根拠は何ですか？
結婚相談所さまは、入会していただければ入会金・月会費・成婚料でお一人あたり数十万円規模の収益になります。その入り口となる面談1件が ¥5,000、しかも予約が成立したときだけ。成婚1組の収益と比べると桁が2つ3つ違います。予約が来なければゼロ円、来たら ¥5,000 という成果報酬なので、費用対効果は自然と担保されます。
Q
キャンセルになっても料金が発生するのはおかしくないですか？
予約成立から24時間以内のキャンセルは全額返金します。24時間を過ぎた場合は、Kinda として集客・送客の役務を果たしている点から、原則として送客料が発生します。ただし、相談所都合またはユーザーの日程変更で別日の面談が了承された場合は返金対応します。
Q
まだ始まったばかりのサービスで、ユーザーがいるんですか？
現在はSEOを軸にユーザー集客を進めています。正直に申し上げると、今はシード段階で、掲載相談所・ユーザー数ともにこれから増やしていくフェーズです。だからこそ今が早期参加のタイミングで、口コミが積み上がるほど先行優位になります。初期費用ゼロなので、リスクを抑えて参加いただけます。
