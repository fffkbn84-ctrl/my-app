1. 型を選ぶ

主力はふびん型（事業計画v1）。週の目安：ふびん3本・淡々2本・ひとこと1〜2本・その他1本。型を選ぶと下の「3. 中身」に完成形の下書きが入ります。案が尽きたらClaudeに「◯◯型の題材を5個」と頼めば補充できます。

タイトル（フック1行目）の型 — 13字以内・カット1に0秒から表示
  • 数字言い切り型（淡々型）：「初回面談で聞かれること3つ」。前置きなし、数字が約束になる
  • 場面切り取り型（ふびん型）：「予約フォームで、30分」。説明せず場面だけ。読点で間を作る体言止めが今の主流
  • セリフ型（ふびん・じぶんツッコミ型）：「『今週こそ』が、3週目」。第一人称の心の声
  • 途中で止める型（ひとこと型）：「口コミは、」。続きを見ないと気持ち悪い状態を作る
  • 正直告白型（ぜんぶ見せる型）：「口コミ、最初は0件でした」。弱みから入ると信頼される
  • 疑問型（翻訳型）：「『仮交際』って、何が仮？」。読者の頭の中の疑問をそのまま置く
  • 逆説型（救いの行）：「迷えるのは、いい兆し」。常識をそっと裏返す

使わないもの：「知らないと損」「9割が間違い」等の煽り数字、「見ないでください」等のカリギュラ型。トレンドではあるがKindaの焦らせない憲法に反する。差別化としても「静かなフック」で通す。

なぜこの6つの型なのか

依頼された5つの発信源（まめきちまめこ・可哀想に!・轟ちゃん・サラ・アートゥーン!）を調査し、「クスッと笑えて、第三者が傷つかない」を成立させている共通構造を抽出しました。結論はこの5つです。

  • 笑いの矛先は「自分」か「自作キャラ」だけ。 全員、他人や視聴者や業界を落とさずに笑いを作っている
  • 定型の反復。 同じ型を繰り返すから「いつもの」として読まれ、数字の比較（検証）もできる
  • かわいい×リアルのギャップ。 整えた広告っぽさを消し、本音に触れても重くならない
  • 失敗を隠さない誠実さが信頼になる。 きれいな部分だけ見せる発信は保存されない
  • 「これ私だ」と重ねられる余白。 具体的な場面で書き、説教や結論で締めない

発信源ごとの分析

まめきちまめこニートの日常 → じぶんツッコミ型

ライブドアブログ最高記録級の日常マンガブログ。家族・犬・猫と自分の日常だけで月間数千万PV。

効いている構造
日常の数十秒を「フリ → 予想外の展開 → セルフツッコミ」で切り取る短い定型。登場人物を身内に固定し、笑いの矛先がつねに自分側。猫の擬人化が現実の毒を抜くクッションになっている。
Kindaが借りるもの
矛先を自分に固定する構造／オチのあとに余韻をひとこと置く呼吸／毎回同じ型で書く反復。
借りないもの
毒っ気の強いボケ。Kindaトーンでは脱力方向に倒す。

可哀想に!（おぱんちゅうさぎ・んぽちゃむ）→ ふびん型

「不憫かわいい」でZ世代を掴んだキャラクター。かわいい見た目×報われない日常のギャップ。

効いている構造
キャラが視聴者の代わりに凹み、報われない目に遭う。見る人は誰かを笑うのではなく「これ私だ」と笑いながら救われる。実在の人が一切傷つかない笑いの完成形。
Kindaが借りるもの
クレイふうか像を「代わりに凹むキャラ」にする。検討層の不安をキャラに背負わせれば、誰も落とさずに共感が作れる。クレイの質感はKindaの差別化資産そのもの。
借りないもの
「可哀想に!」という突き放し。Kindaは最後にそっと救いのひとことを置く（焦らせない憲法）。

整形アイドル轟ちゃん → ぜんぶ見せる型

整形のリアル（ダウンタイム・失敗談まで）を隠さず発信して信頼を獲得したYouTuber。

効いている構造
語りにくい領域の当事者開示。成功だけでなく失敗も全部見せるから信じられる。視聴者の相談に答える双方向で、一次情報が湧き続ける。
Kindaが借りるもの
運営と取材の裏側を失敗込みで見せる誠実さ。「中を覗いてから決めたい」という検討層の本音（柱B）への回答と同じ構造。
借りないもの
過激さ・ビフォーアフター的な煽り。開示するのは自分たちの裏側だけで、相談所や会員の内情を暴露しない。

コスメヲタちゃんねるサラ → 淡々型

無駄な解説なし・淡々とテンポよく進む正直レビューで登録者100万人超の美容系YouTuber。

効いている構造
感情を盛らず事実を並べる。誇張しないから信頼され、あとで見返すために保存される。同じ素材をショート3媒体（YouTube・TikTok・リール)に同時展開して量産。
Kindaが借りるもの
事実を3つ淡々と並べる保存狙いの型。1つの素材をIG・X・noteへ同時展開する運用（/sns-packと同じ思想）。
借りないもの
比較の言い切り。相談所をランク付け・比較しない（憲法）。

アートゥーン! → 翻訳型

東京藝大出身の8人組。「芸術をもっと身近に」を掲げ、専門知を楽しく翻訳して急成長。

効いている構造
敷居の高い専門領域を、専門家本人がゆるく日常語にほどく。専門性×ゆるさのギャップが「知らない世界を覗く楽しさ」になる。
Kindaが借りるもの
カウンセラーの専門知・業界用語を日常語に翻訳する型。Kinda voicesの取材素材（引用ブロック）がそのまま原料になる。
借りないもの
内輪ノリ。メンバー文化の面白さはKindaには移植できないので、翻訳の構造だけ借りる。

Kinda voices 構成7（引用ブロック）→ ひとこと型

これは外部由来ではなく、voices記事構成の「印象に残っているひとこと（SNS拡散設計）」を独立させた型。

構造
業界の常識をそっと外す言葉を主役に置き、余計な説明をしない。まめこの「余韻」とサラの「盛らなさ」を掛け合わせた、いちばん静かな型。
注意
引用者の表記は必ず同意範囲内で。CTAは固定文「〇〇さんの話を、直接聞いてみる。」

安全設計（なぜこのチェックなのか）

  • 「〜な人いるよね」を禁止する理由：あるあるの主語を他人にした瞬間、活動中の誰かが自分を笑われたと感じる。主語を自分かクレイふうか像にすれば、同じ共感を誰も傷つけずに作れる（おぱんちゅ構造）
  • 焦らせ・比較の禁止：Kinda憲法。笑いに紛れても「早くしないと」「周りは」は使わない
  • 「相手」「ふたり」で書く：LGBTQ+包摂。笑いのネタでも異性・男女を前提にしない
  • 絵文字は区切りのみ：SNSキャプションは最小限許容（2026-06-08細則）。感情表現は言葉でやる
  • 同意範囲の確認：voices・story由来の素材は、実名・相談所名・写真の同意項目を出す前に確認（ステマ規制と信頼の核）

リール形式の調査（2026-07・仮説）

「リール優勢」の先を調査した結論。各型の設計図（つくるタブでIG×リールを選ぶと表示）はこの5原則に基づく。

  • 冒頭1.5〜2秒がすべて。最初の3秒で約半数が離脱し、3秒残存率60%超のリールは40%未満の5〜10倍届く。1フレーム目にテキストフック、2秒以内に2カット目（2カットの法則）
  • 尺は目的で使い分け。ループ狙い7〜15秒（完了率で伸ばす）／標準15〜30秒／教育・保存狙い30〜60秒。完了率80%が目標線
  • 無音視聴前提の全編テロップ。音声はふうかの声で二重化（顔なしでも声は信頼をつくる資産）
  • 作り込まない"raw感"が優勢。広告っぽい高品質より、粗さが届く。手元のスマホ写真と生成クレイ画像の混在で問題ない
  • 静止画スライド式リールで量産。画像1枚2〜5秒＋テロップで成立（イラスト・漫画系アカウントの定石）。まめこ式4コマはこの形式にそのまま載る
  • Kindaの制作制約と噛み合う。クレイふうか像は生成AI画像で、枠は1日5〜8枚（ChatGPT無料プラン）。画像の動画化は能力が低いため1本につき1カットまで。静止画スライド中心の設計はこの制約下でちょうど回る（ふびん型3〜4枚＝1日1〜2本分、ひとこと型は1枚）

アルゴリズムの上位指標は保存・シェア・完了率。バズよりこの3つを分析タブで追う。すべて仮説なので、2週間の実測で設計図を更新する。

このサイトの使い方

  • ホーム画面に追加：iPhoneのSafariでこのページを開き、共有ボタン → 「ホーム画面に追加」。次からワンタップで開けます
  • 毎日の流れ：ネタ帳でシャッフル → つくるで型を埋める → 安全チェック → Notion用コピーを投稿カレンダーに貼る
  • 週次：分析タブに数字を記録 → CSVコピー → /sns-review に渡す
  • 下書き・ネタ・記録はこの端末のブラウザ内に保存されます（別端末とは共有されません）
