# SNS計測CSV の書き方

`/sns-review` Skill が読む実績データ。月ごとに `YYYY-MM.csv` を1ファイル（例: `2026-07.csv`）。
きれいに書けない日は、チャットに数字を貼れば Claude が整形して追記する（それで十分）。

## 列定義

| 列 | 内容 | 例 |
|---|---|---|
| date | 投稿日 | 2026-07-02 |
| media | X / IG / note | X |
| pillar | A天気 / B相談所リアル / C問いかけ / Dつくる日記 / E告知 / F中の人（noteは記事、IGはreel/carousel等でも可） | C |
| title | 投稿の冒頭や識別できる短いメモ | 雨の日のお見合い、どうしてる？ |
| impressions | 表示回数 | 1240 |
| engagements | いいね＋リプ＋RT＋保存の合計（分けられるなら memo に内訳） | 38 |
| profile_visits | プロフィールクリック | 12 |
| link_clicks | リンククリック（リプのリンク含む） | 4 |
| follows | この投稿経由のフォロー（不明なら空欄） | 1 |
| memo | 時間帯・気づき・内訳など自由 | 21時投稿。自リプにコラムURL |

## 記録のルール

- **毎日書かなくてよい。週1回、Xのアナリティクスを見ながらまとめて書く**（続くことが最優先）
- 不明な数字は空欄でよい（0と空欄を区別する）
- note は月1回、記事ごとのビュー・スキ・読了率（分かれば）を同じ形式で
- 数字はふうかの手入力が正。Claude は追記整形のみ行い、既存行を書き換えない
