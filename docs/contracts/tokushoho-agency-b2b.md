# 特定商取引法に基づく表記（結婚相談所さま向け・送客サービス）

> **このドキュメントの位置付け**
> Kinda の収益は、結婚相談所（事業者）から受け取る **送客手数料 ¥5,000/件（成果報酬）** です。
> この課金は Stripe を通じてカウンセラー管理画面（`futarive-counselor`）側で行われるため、
> **Stripe 審査・事業者向けの特定商取引法表記**として、このページを **カウンセラー管理画面側に公開**する想定です。
>
> - ユーザー（消費者）向けの特商法表記は別途 `src/app/tokushou/page.tsx`（ユーザーは無料）にあります。本表記は**事業者向け**です。
> - **［　］のうち、ドメイン名・法人名・代表者名・所在地・電話番号は法人設立後に実値を挿入**。それ以外は確定値で記載済み。
> - 公開前に顧問弁護士のレビューを通すこと（特に事業者間契約と特商法の適用関係）。
> - 内容は送客契約書（`docs/contracts/agency-agreement.md`）第6条および CLAUDE.md §8・§12 と整合させること。変更時は三者を併せて更新する。

### 返金ポリシー（送客手数料）

| ケース | 取り扱い |
|---|---|
| 予約成立から **24時間以内**のキャンセル（当事者の都合を問わず） | **全額返金** |
| 24時間経過後・**エンドユーザー都合**のキャンセル | 返金なし・課金確定 |
| 24時間経過後・**相談所都合**のキャンセル → 別日程を合意 | いったん返金 → 別日程の予約成立時に再課金 |
| エンドユーザーの**日程変更**を相談所が承諾 | いったん返金 → 別日程の予約成立時に再課金 |
| 上記以外で面談が実施されなかった場合 | 返金なし・課金確定 |

### その他

- エンドユーザー（消費者）向けの表記は、ユーザーサイトの「特定商取引法に基づく表記」をご参照ください。
