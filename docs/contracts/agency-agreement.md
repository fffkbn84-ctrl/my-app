## 契約書本文

[会社名]（以下「甲」という）と〔結婚相談所オーナー氏名／法人名〕（以下「乙」という）は、甲が運営するプラットフォーム「Kinda ふたりへ」（kinda-futari.app）（以下「本サービス」という）における乙の事業（以下「乙の事業」という）の掲載および乙への送客に関し、以下のとおり契約（以下「本契約」という）を締結する。

### 第1条（目的）

本契約は、甲が乙の事業（結婚相談所およびその所属カウンセラーの提供する婚活サポート役務）を本サービス上に掲載し、エンドユーザー（本サービスを利用する個人）から乙への送客（面談予約の成立およびその実施）を媒介することを目的とする。

### 第2条（定義）

本契約において使用する用語の定義は以下のとおりとする。

1. 「本サービス」：甲が運営する Kinda ふたりへ（kinda-futari.app）およびこれに関連するすべての Web サービス・管理ツール
2. 「公開プラットフォーム」：エンドユーザーが閲覧する `https://kinda-futari.app`
3. 「カウンセラー管理画面」：乙および乙の所属カウンセラーが利用する管理画面 `https://futarive-counselor.vercel.app`（または甲が指定するドメイン）
4. 「エンドユーザー」：本サービスを利用する個人（婚活を検討する者を含む）
5. 「相談所オーナー」（または「オーナー」）：乙（本契約の締結相手・契約上の責任主体）
6. 「配下カウンセラー」：乙が運営する相談所に所属するカウンセラー（個人事業主／業務委託／雇用契約のいずれを問わない）であって、本サービス上に乙の招待によりプロフィールが掲載される者
7. 「送客」：エンドユーザーが本サービス上の予約機能を通じて乙または配下カウンセラーへの面談予約を成立させ、当該面談が現実に実施されたこと
8. 「利用規約」：甲が本サービス上に掲示する「Kinda カウンセラー管理画面 利用規約」（`https://futarive-counselor.vercel.app/terms`）

### 第3条（本契約の効力発生）

本契約は、甲乙双方が記名押印または電子署名した日に効力を生じるものとする。乙が本サービスのカウンセラー管理画面において利用規約に同意した日と本契約の締結日が異なる場合、いずれか遅い日をもって本契約の効力発生日とする。

### 第4条（甲の役務）

1. 甲は、乙および配下カウンセラーが本サービスに登録した情報（プロフィール・料金プラン・対応エリア・自己紹介文・画像等）を公開プラットフォームに掲載する。
2. 甲は、本サービス上でエンドユーザーが乙または配下カウンセラーに対して面談予約を行えるよう、予約システムおよび関連通知機能を提供する。
3. 甲は、エンドユーザーから本サービスを通じて取得した予約情報（氏名、ふりがな、メールアドレス、電話番号、希望日時、アンケート回答等）を、乙および配下カウンセラーに対し面談実施に必要な範囲で提供する。
4. 甲は、本サービスの安定運用のために必要な技術的措置（サーバ運用、セキュリティ対策、システム改修等）を講じる。
5. 甲は、本サービスを通じた送客の数および成果について、いかなる保証も行わない。

### 第5条（乙の役務）

1. 乙は、利用規約に従い、本サービス上に乙の事業に関する情報を正確に登録し、最新の状態を維持する義務を負う。
2. 乙は、本サービスを通じて成立した送客（面談予約）に対し、エンドユーザーに対して誠実にカウンセリング・婚活サポートを提供する。
3. 乙は、配下カウンセラーが利用規約および本契約を遵守するよう、合理的かつ適切な監督を行う義務を負う（利用規約第12条の2 参照）。
4. 乙は、本サービスを経由しない面談予約・連絡（SNS、個別 LINE、電話その他の本サービス外の連絡手段）について、送客手数料の発生を回避する目的でエンドユーザーを誘導・移管してはならない（利用規約第6条第6項・第9条参照）。

### 第6条（送客手数料）

1. 甲は、本サービスを通じて送客が成立した場合、乙に対し、送客 1 件あたり **金 5,000 円（税別）** の送客手数料を請求する。
2. 「送客の成立」とは、エンドユーザーが本サービスの予約システムを通じて乙または配下カウンセラーに対し面談予約を行い、かつ当該面談が現実に実施された時点をいう。**面談が予約後にキャンセル・無断不参加となった場合は、送客手数料は発生しない。**
3. 甲は、毎月末日を締日とし、当月締分の送客手数料を翌月末日までに乙に対し請求書を発行する。乙は、当該請求書の発行日から **30 日以内** に、甲の指定する銀行口座に振込により支払うものとする。振込手数料は乙の負担とする。
4. 乙が支払期日までに送客手数料を支払わない場合、乙は、甲に対し、支払期日の翌日から完済日まで **年 14.6％** の割合による遅延損害金を支払うものとする。
5. 乙が、送客手数料の発生を回避する目的をもって、本サービスを通じて知り得たエンドユーザーとの面談予約・連絡を本サービス外の連絡手段に移管・誘導した場合、甲は、当該行為がなければ本サービスを通じて成立し得たであろう送客手数料相当額（甲が合理的に算定するものとする）の支払いを乙に対し請求できる。
6. 本契約に基づき乙から甲に支払う金額には、別途消費税および地方消費税が課されるものとする。

### 第7条（初期費用および月額固定料金）

本サービスの基本機能（プロフィール掲載、予約管理、口コミ閲覧、ダッシュボード等）の利用について、甲は乙に対し、初期費用および月額固定料金を請求しない。甲が将来的に有料のプレミアム掲載オプションを提供する場合、その料金・条件は別途合意するものとする。

### 第8条（口コミ）

1. 本サービス上で乙および配下カウンセラーに関する口コミは、本サービスを通じて面談を完了したエンドユーザーまたは甲が代理掲載する者（「代理掲載」バッジが付されたもの）に限り投稿される。
2. 乙は、口コミの内容について、本サービスを通じて甲所定の方法により異議申立、削除請求、返信投稿を行うことができる。甲は、申立内容を確認のうえ、本サービスの運営方針およびプライバシーポリシーに従い、合理的な範囲で対応する。
3. 乙および配下カウンセラーは、口コミ投稿者に対し、報酬・便益・割引等を提供する見返りに特定の評価・内容での口コミ投稿を依頼してはならない（ステルスマーケティング・やらせ口コミの禁止）。

### 第9条（個人情報の取り扱い）

1. 乙および配下カウンセラーは、本サービスを通じて取得したエンドユーザーの個人情報を、面談実施および本サービスの目的の達成に必要な範囲においてのみ利用するものとする。
2. 乙および配下カウンセラーは、エンドユーザーの個人情報を、個人情報の保護に関する法律その他関係法令に従い、適切に管理する。
3. 乙は、配下カウンセラーが本契約を脱退・退職した場合、当該配下カウンセラーが本サービスを通じて取得したエンドユーザーの個人情報を以後保持しないよう必要な指示を行う。
4. 甲による個人情報の取り扱いは、別途定めるプライバシーポリシー（`https://futarive-counselor.vercel.app/privacy`）に従う。

### 第10条（禁止事項）

乙および配下カウンセラーは、本サービスの利用にあたり、利用規約第9条に定める禁止事項に該当する行為をしてはならない。特に以下の行為を厳に禁ずる。

### 本テンプレートの想定運用フロー

1. **営業初回接触**：相談所オーナーに本契約書のドラフト（実値挿入版）と Kinda 概要資料を送付
2. **質問対応・条件交渉**：必要に応じて条項調整（送客手数料の額・支払サイト等）
3. **契約締結**：書面（記名押印・PDF スキャン）またはクラウドサイン等の電子契約
4. **契約締結後、運営側で実施**：
   - futarive-admin から agencies / counselors（オーナー本人のレコード）を作成
   - invite_token を発行し、招待 URL `https://futarive-counselor.vercel.app/claim?token=UUID` をオーナーへ送付
5. **オーナー操作**：
   - 招待 URL から /claim で signup（利用規約に明示的同意）
   - その後、agencies.owner_user_id を運営側が手動でオーナーの auth.uid() にセット
6. **以降の運用**：オーナーが配下カウンセラーを招待（同じ /claim フロー）

### 顧問弁護士レビュー時のチェックポイント

- 第6条の送客手数料の発生条件（面談実施が要件）の文言が明確か
- 第6条第5項の「送客手数料相当額」の算定根拠が損害賠償として無効になっていないか
- 第12条の2（オーナー監督責任）の連帯責任条項が法的に有効か
- 第14条（即時解除）の各号が公序良俗違反にならない範囲か
- 個人情報保護法および特定電子メール法との整合性
- 消費者契約法は本契約には適用されない（事業者間契約）旨を明確化するか検討
